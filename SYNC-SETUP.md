# Shadowloop 跨裝置同步設定（Google Apps Script + Sheets）

只要設定一次，之後電腦和手機的練習紀錄就會自動合併同步。

## 一、建立後端（約 5 分鐘）

1. 到 <https://sheets.new> 建立一個新的 Google 試算表，名字隨意（例如 `Shadowloop Sync`）。
2. 上方選單 **擴充功能 → Apps Script**。
3. 刪掉編輯器裡預設的內容，把本專案 `apps-script/Code.gs` 的全部內容貼進去，按 **儲存**。
4. 右上角 **部署 → 新增部署**：
   - 類型選 **網頁應用程式**
   - 「執行身分」選 **我**
   - 「具有存取權的使用者」選 **任何人**（Anyone）
   - 按 **部署**，第一次會要求授權，一路允許即可。
5. 複製部署完成後給你的 **網頁應用程式 URL**（`https://script.google.com/macros/s/…/exec`）。

## 二、在 App 裡開啟同步

1. 打開 Shadowloop → **Home** 分頁 → 最下面設定卡片。
2. 把剛才的 URL 貼到 **Cloud sync** 的輸入框，按 Enter（會立刻做第一次同步）。
3. 在手機瀏覽器打開同一個 Shadowloop 網址（GitHub Pages），貼上**同一個 URL**，按 **Sync now**。

完成。**之後完全自動，不用手動按任何按鈕**：

- **回到 App（開啟、或從其他 App 切回來）**：自動拉取雲端最新資料並合併。
- **練習 / 存 clip 之後**：約 5 秒自動推上雲端。
- **切走或關閉分頁時**：把還沒推的變更立刻補推上去。
- 換裝置時什麼都不用做——在電腦練完直接關掉，拿起手機打開就是最新進度。
- 萬一網路不穩想確認，可以隨時按 **Sync now** 手動同步（平常用不到）。

## 合併規則（兩台裝置都改過也不會壞）

- **Clips**：同一個 clip 以「最後練習時間較新」的版本為準；不同 clip 直接聯集。
- **每日紀錄**：同一天同一個 clip 取次數較大的值（不會重複累加）。
- **資料夾**：取聯集。

## 注意事項

- 這個 URL 等同於你的資料鑰匙，**不要公開分享**（知道 URL 的人可以讀寫你的練習資料）。
- 若之後修改了 `Code.gs`，要用 **部署 → 管理部署 → 編輯 → 新版本** 重新部署，URL 才會生效更新。
- 試算表裡 `shadowloop_data` 工作表是資料本體（JSON 分段存放），不要手動編輯。
